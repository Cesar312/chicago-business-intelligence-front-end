Chicago Business Intelligence

Front-End Demonstration

Project Phase 4 is the final phase of the project. The project is to create a business intelligence dashboard for the City of Chicago. The dashboard provides key metrics to make data-driven decisions. The dashboard was created using Tableau. The data for the dashboard was obtained from the City of Chicago Data Portal. The data loaded and transformed using a Golang-Microservice and stored on a PostgreSQL cloud database hosted by Google. The data is made available publicly by a web server in Google Cloud.
